# Vision: vision-from-idea

> Generated by Concert 3 Init

## Problem Statement
concert3 takes a high level idea description or file and expands and converts the idea into a proper vision document. This could be considered a new step at the begining of the workflow so that the vision document can also have a review process and a specialist agent. If the idea was from a file, that file is no longer used. The main goal is to have concert create a strong vision document for the user and be able to have a review process to improve the quality of the visin.

## Goals
Given a high level idea statement or file, comprehensively expand it, including web based research, into a strong vision to be the foundation for a concert mission. Adding the ability to review and revise the vision document will improve the end result by having a clear starting point and solid direction from the first step.

## Non-Goals
- Does not include detailed requirements, architecure, or ux unless explicitly asked for in the idea. 

## Success Criteria
- [ ] Must create a vision document with all the proper formatting and necessary sections populated.
- [ ] Must be able to Revise on the Vision stage of the workflow.
